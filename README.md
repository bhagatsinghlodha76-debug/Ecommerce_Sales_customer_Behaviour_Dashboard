E-Commerce Sales & Customer Behaviour Analytics Dashboard
Project Overview

This Power BI project analyzes E-Commerce Sales Performance and Customer Behaviour using interactive dashboards.

The dashboard helps businesses understand:

Total Sales
Customer Acquisition Channels
Customer Demographics
Product Performance
Payment Methods
Regional Sales Distribution
Order Status Analysis
Monthly Sales Trends
